Uh Oh!

Uh Oh! Challenge - Hint (NOT USED) ![Uh Oh! Challenge Hint](/images/patriotctf_2023/Uh_Oh_Hint.jpg)

Write-Up

  1. Downloaded the file provided (rockyou.zip)
    • Unzipped the file.
    • Zip file contained a modified rockyou.txt file.
  2. Opened up browser and visited the website referenced:
    • https://en.wikipedia.org/wiki/North_American_Numbering_Plan#Modern_plan
      • Reviewed the information provided on NANP requirements. Allowing to rule out any digits and formatting not allowed.
  3. Created a regex expression based on NANP and provided phone format.
    • Phone Format: (NPA) NXX-XXXX
    • Regex: '^\([2-9][0-8][0-9]\) [2-9][0-9]{2}-[0-9]{4}$'
      • N character can’t have the 1 digit.
      • P character can’t have the 9 digit.
      • Characters A and X are clear for digits 0 to 9.
  4. Opened Terminal to folder holding the modified rockyou.txt file.
    • Used grep command to find the regex expression and to print the line info.
    • Results from the command were both sections of the Flag.

FLAG:

PCTF{7731484_4043037283}
Flag is not case sensitive.
