Write-Up

  1. Opened up the Terminal and ran dirb command on the website:

    dirb http://chal.pctf.competitivecyber.club:5999
    • Scan results provided the following 2 URLs:
      • /index.html (main website)
      • /robots.txt (file located on website)

  2. Opened up Burpsuite and used Proxy browser with Interceptor turned on.

    • Entered URL listed on the challenge page.
      • http://chal.pctf/competitivecyber.club:5999/
      • Clicked Forward.
    • Website displays Flag part 1.
    • Entered the additional URL found with dirb command.
      • http://chal.pctf/competitivecyber.club:5999/robots.txt
      • Clicked Forward.
    • Website displays Flag part 3.

  3. On Burpsuite, selected HTTP History under Proxy tab:

    • Cicked on “URL/” and changed Response to Raw.
      • Code Review: Line 26 Comment displays Flag part 2.
    • Cicked on next History item “/script1.js” and changed Response to Raw.
      • Code Review: Line 11 console.log displays Flag part 4.
    • Cicked on next History item “/favicon.ico” and viewed Request Raw.
      • Code Review: Line 8 Cookie displays Flag part 5.

  4. Combined the separate flags found into one total Flag.

    • Flag 1/5 - PCTF{Hunt
    • Flag 2/5 - 3r5_4n
    • Flag 3/5 - D_g4tH3
    • Flag 4/5 - R5_e49
    • Flag 5/5=e4a541}

FLAG:

PCTF{Hunt3r5_4nD_g4tH3R5_e49e4a541}
Flag is case sensitive.
